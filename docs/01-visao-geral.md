# Visão Geral

## Introdução

O [CodeFlex](../index.html) é uma plataforma gratuita de ensino de programação criada no contexto da curricularização da extensão.

O projeto tem como objetivo oferecer acesso ao aprendizado de programação para iniciantes, utilizando uma abordagem prática e interativa, com elementos de **gamificação**, como sistemas de XP, minijogos e diversos outros recursos voltados para tornar o aprendizado mais envolvente.

## Objetivo

O objetivo do [CodeFlex](../index.html) é proporcionar uma plataforma acessível para o aprendizado de programação, permitindo que pessoas iniciantes desenvolvam conhecimentos técnicos de forma gratuita, organizada, interativa e motivadora.

## Motivação

O aprendizado de programação pode representar uma barreira para muitas pessoas devido à falta de materiais gratuitos e organizados, às dificuldades enfrentadas no início da aprendizagem e à ausência de acompanhamento durante os estudos.

O [CodeFlex](../index.html) busca reduzir essas barreiras oferecendo uma experiência de aprendizado simples, prática e motivadora.

## Público-alvo

A plataforma é destinada principalmente a:

* Pessoas iniciantes em programação.
* Estudantes interessados na área de tecnologia.
* Pessoas que desejam iniciar no desenvolvimento de software.
* Qualquer membro da comunidade interessado em aprender programação gratuitamente.

## Proposta

O CodeFlex oferece uma plataforma com cursos, exercícios práticos e recursos interativos para auxiliar no desenvolvimento das habilidades de programação.

A utilização de elementos de gamificação busca aumentar o engajamento dos usuários, tornando o aprendizado mais dinâmico, intuitivo e agradável.

## Impacto esperado

O projeto busca ampliar o acesso ao conhecimento tecnológico, contribuindo para a formação de novos estudantes na área de programação e incentivando o desenvolvimento de habilidades digitais na comunidade.

## Escopo inicial

A primeira versão da plataforma contará com:

* Cadastro de usuários.
* Trilhas de aprendizado.
* Conteúdos introdutórios.
* Exercícios práticos.
* Sistema de progresso.
* Sistema de gamificação.

## Relação com a extensão universitária

O projeto busca aproximar o conhecimento desenvolvido no ambiente acadêmico da comunidade externa, oferecendo uma ferramenta gratuita de ensino de programação e contribuindo para a democratização do acesso ao conhecimento tecnológico.
